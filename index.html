Definitions:

HTML: HTML stands for HyperText Markup Language. It gives structure to a webpage using tags like headings, paragraphs, images, and forms.

CSS: CSS means Cascading Style Sheets. It is used to style HTML pages such as colors, fonts, layout, spacing, and responsiveness.

JavaScript: JavaScript adds interactivity to webpages. It makes buttons work, forms validate, animations run, and pages dynamic.

API: API stands for Application Programming Interface. It allows two software programs to communicate and exchange data.

Server: A server is a powerful computer that stores data, websites, and applications, and delivers them to users when requested.

Client: A client is the user's device or browser that sends requests to the server and displays the results.

Frontend: The visible part of a website such as design, layout, buttons, forms, visuals. Built using HTML, CSS, and JavaScript.

Backend: The behind-the-scenes logic such as databases, authentication, servers, and processing.

IP Address: A unique number assigned to every device on the internet. It tells where data should be sent and received.

DNS: DNS stands for Domain Name System. It translates domain names (like google.com) into IP addresses.

Algorithm: An algorithm is a step-by-step method to solve a problem efficiently.

Compiler: A compiler converts high-level code (C, C++, Java) into machine code that a computer can execute.

Database: A structured system that stores, organizes, and retrieves data. Example: MySQL, MongoDB.

Operating System: Software that manages hardware and lets you run programs. Examples: Windows, Linux, Android.
